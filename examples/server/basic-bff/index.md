# Basic BFF Examples

Backend-for-Frontend pattern examples using server composables.

## What is BFF?

The BFF (Backend-for-Frontend) pattern creates a dedicated backend layer between your frontend and external APIs. This provides:

- **Security** - Hide API keys and sensitive logic
- **Transformation** - Shape data for your UI needs
- **Aggregation** - Combine multiple API calls
- **Caching** - Server-side response caching
