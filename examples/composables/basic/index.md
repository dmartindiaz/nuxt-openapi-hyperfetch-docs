# Basic Examples

Learn the fundamentals of using generated composables with these basic examples.

## Examples

- **[Simple GET Request](./simple-get)** - Fetching data from an API endpoint
- **[Path Parameters](./path-parameters)** - Using dynamic path parameters
- **[Query Parameters](./query-parameters)** - Adding query parameters to requests
- **[POST Request](./post-request)** - Sending data to create or update resources

## What You'll Learn

These examples cover:

- Basic usage of `useFetch*` composables
- Handling loading states and errors
- Working with TypeScript types
- Request parameters (path, query, body)
