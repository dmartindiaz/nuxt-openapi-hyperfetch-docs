# Callback Examples

Learn how to use per-request callbacks to handle specific behaviors for individual API calls.

## Examples

- **[Error Toast](./error-toast)** - Show toast notifications on errors
- **[Success Navigation](./success-navigation)** - Navigate after successful requests
- **[Request Logging](./request-logging)** - Log request details for debugging
- **[Loading Spinner](./loading-spinner)** - Show/hide loading indicators

## What You'll Learn

- Using `onRequest`, `onSuccess`, `onError`, and `onFinish` callbacks
- Handling specific use cases per request
- Combining callbacks with global handlers
- Best practices for callback usage
